常見問題

什麼是 WhatsApp Coexistence？
WhatsApp Coexistence 是一項功能，可讓企業在同一個電話號碼上同時使用 WhatsApp Business App 和 WhatsApp Cloud API。訊息會在兩個平台之間即時同步，讓您在利用 API 的自動化和規模的同時，保持行動應用程式的個人化服務。
我需要變更我的電話號碼嗎？
不用！WhatsApp Coexistence 的關鍵優勢之一是您可以保留現有的電話號碼。您不需要取得新號碼或變更現有號碼。QR 碼為基礎的導入程序會將您目前的 WhatsApp Business App 號碼連結到 Cloud API。
我會遺失聊天記錄嗎？
不會，您可以在導入期間保留最多 180 天的聊天記錄。當您在內嵌註冊程序中將 WhatsApp Business App 連結到 Cloud API 時，您可以選擇匯入過去六個月的對話記錄和聯絡人。但這必須在完成導入程序後的 24 小時內完成。
定價如何運作？
WhatsApp Coexistence 啟用混合定價模式。從 WhatsApp Business App 手動傳送的訊息完全免費。透過 Cloud API 傳送的訊息（包括自動訊息、範本和廣播）遵循 Meta 的對話為基礎定價模式。這讓您可以透過使用免費的應用程式訊息進行個人一對一對話，以及使用 API 訊息進行規模和自動化來優化成本。
技術限制是什麼？
主要技術限制是：1）共存帳戶的傳輸量限制為每秒 20 訊息（MPS），相較於獨立 API 帳戶的更高限制；2）您必須至少每 13-14 天打開一次 WhatsApp Business App 以維持同步連線；3）某些應用程式功能已停用，包括群組聊天同步、消失訊息、一次性檢視媒體和訊息編輯/撤銷。
在所有國家都可用嗎？
不，WhatsApp Coexistence 正在分階段推出。截至 2026 年 1 月，某些地區（包括奈及利亞和南非）無法使用。這些國家的企業目前不符合資格，需要在獨立應用程式或獨立 API 之間選擇。
遷移需要多長時間？
遷移程序旨在快速且無縫。QR 碼為基礎的內嵌註冊流程通常需要 5-10 分鐘才能完成。如果您選擇匯入聊天記錄，這將在背景進行。整個過程可以在 15 分鐘內完成，不會中斷您目前的 WhatsApp 運營。
如果我停止使用會怎樣？
您可以隨時停止使用 Cloud API。您的 WhatsApp Business App 將繼續正常使用您的號碼。但是，您將失去對 API 提供的自動化、多Agent功能和 CRM 整合的存取權。在聯繫 Meta 支援並完成新的導入程序之前，您將無法重新啟用該號碼的共存功能。
