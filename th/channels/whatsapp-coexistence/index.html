คำถามที่พบบ่อย

WhatsApp Coexistence คืออะไร?
WhatsApp Coexistence เป็นคุณสมบัติที่อนุญาตให้ธุรกิจใช้ทั้งแอป WhatsApp Business และ WhatsApp Cloud API บนเบอร์โทรศัพท์เดียวกัน พร้อมกัน ข้อความจะถูกซิงค์แบบเรียลไทม์ระหว่างแพลตฟอร์มทั้งสอง ทำให้คุณสามารถรักษาความเป็นส่วนตัวของแอปมือถือไว้ได้ในขณะที่ใช้ประโยชน์จากระบบอัตโนมัติและการขยายขนาดของ API
ฉันจำเป็นต้องเปลี่ยนเบอร์โทรศัพท์ไหม?
ไม่! หนึ่งในประโยชน์หลักของ WhatsApp Coexistence คือคุณสามารถเก็บเบอร์โทรศัพท์ที่มีอยู่เดิมไว้ได้ คุณไม่จำเป็นต้องขอเบอร์ใหม่หรือเปลี่ยนเบอร์ที่มีอยู่ กระบวนการลงทะเบียนผ่าน QR Code จะเชื่อมโยงเบอร์ WhatsApp Business App ปัจจุบันของคุณเข้ากับ Cloud API
ฉันจะสูญเสียประวัติแชทไหม?
ไม่ คุณสามารถรักษาประวัติแชทได้สูงสุด 180 วันระหว่างการลงทะเบียน เมื่อคุณเชื่อมโยง WhatsApp Business App ของคุณเข้ากับ Cloud API ผ่านกระบวนการลงทะเบียนแบบฝัง คุณมีตัวเลือกในการนำเข้าประวัติการสนทนาและรายชื่อติดต่อหกเดือนล่าสุดของคุณ อย่างไรก็ตาม ต้องดำเนินการภายใน 24 ชั่วโมงหลังจากเสร็จสิ้นกระบวนการลงทะเบียน
ระบบการกำหนดราคาทำงานอย่างไร?
WhatsApp Coexistence ช่วยให้สามารถใช้โมเดลการกำหนดราคาแบบไฮบริดได้ ข้อความที่ส่งด้วยตนเองจาก WhatsApp Business App ยังคงฟรี 100% ข้อความที่ส่งผ่าน Cloud API (รวมถึงข้อความอัตโนมัติ เทมเพลต และการส่งแบบกระจาย) จะอยู่ภายใต้โมเดลการกำหนดราคาตามการสนทนาของ Meta ซึ่งช่วยให้คุณสามารถเพิ่มประสิทธิภาพต้นทุนโดยใช้ข้อความแอปฟรีสำหรับการสนทนาแบบ 1:1 ส่วนตัว และข้อความ API สำหรับการขยายขนาดและการทำระบบอัตโนมัติ
ข้อจำกัดทางเทคนิคคืออะไร?
ข้อจำกัดทางเทคนิคหลักคือ: 1) ความสามารถในการประมวลผลถูกจำกัดไว้ที่ 20 ข้อความต่อวินาที (MPS) สำหรับบัญชีที่ใช้โหมดอยู่ร่วมกัน เทียบกับขีดจำกัดที่สูงกว่าสำหรับบัญชี API แบบอิสระ; 2) คุณต้องเปิด WhatsApp Business App อย่างน้อยหนึ่งครั้งทุก 13-14 วัน เพื่อรักษาการเชื่อมต่อซิงค์; 3) คุณสมบัติบางอย่างของแอปถูกปิดใช้งาน ได้แก่ การซิงค์แชทกลุ่ม ข้อความที่หายไป ไฟล์สื่อที่แสดงครั้งเดียว และการแก้ไข/ยกเลิกข้อความ
มีให้บริการในทุกประเทศหรือไม่?
ไม่ WhatsApp Coexistence อยู่ในระหว่างการเปิดตัวแบบเป็นขั้นตอน ณ มกราคม 2026 ยังไม่พร้อมใช้งานในบางภูมิภาค ได้แก่ ไนจีเรียและแอฟริกาใต้ ธุรกิจที่มีเบอร์โทรศัพท์จากประเทศเหล่านี้ยังไม่มีคุณสมบัติเหมาะสมและจะต้องเลือกระหว่างแอปอิสระหรือ API อิสระ
การย้ายข้อมูลใช้เวลานานแค่ไหน?
กระบวนการย้ายข้อมูลได้รับการออกแบบให้รวดเร็วและไร้รอยต่อ กระบวนการลงทะเบียนแบบฝังผ่าน QR Code โดยทั่วไปใช้เวลา 5-10 นาทีในการดำเนินการให้เสร็จสมบูรณ์ หากคุณเลือกที่จะนำเข้าประวัติแชท ก็จะเกิดขึ้นในเบื้องหลัง กระบวนการทั้งหมดสามารถดำเนินการให้เสร็จสิ้นภายใน 15 นาทีโดยไม่มีการหยุดชะงักใดๆ ต่อการดำเนินงาน WhatsApp ที่กำลังดำเนินอยู่ของคุณ
เกิดอะไรขึ้นหากฉันหยุดใช้งาน?
คุณสามารถหยุดใช้ Cloud API ได้ตลอดเวลา แอป WhatsApp Business ของคุณจะยังคงทำงานตามปกติด้วยเบอร์ของคุณ อย่างไรก็ตาม คุณจะสูญเสียการเข้าถึงระบบอัตโนมัติ คุณสมบัติหลายตัวแทน และการรวม CRM ที่ API ให้ไว้ คุณจะไม่สามารถเปิดใช้งานโหมดอยู่ร่วมกันสำหรับเบอร์นั้นได้อีกจนกว่าคุณจะติดต่อฝ่ายสนับสนุนของ Meta และดำเนินกระบวนการลงทะเบียนใหม่ให้เสร็จสมบูรณ์
