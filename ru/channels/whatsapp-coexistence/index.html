Что такое WhatsApp Coexistence?

WhatsApp Coexistence - это революционная функция, которая позволяет предприятиям использовать как приложение WhatsApp Business, так и WhatsApp Cloud API на одном и том же номере телефона, одновременно.

✗ До сосуществования

  • • Потеря истории чата при переходе на API
  • • Изменение номера телефона или получение нового
  • • Выбор между личным приложением или корпоративной автоматизацией
  • • Нарушение отношений с клиентами во время перехода

✓ После сосуществования

  • ✓ Сохранить тот же номер телефона
  • ✓ Синхронизировать 180 дней истории чата
  • ✓ Использовать приложение и API одновременно
  • ✓ Миграция без перебоев

Часто задаваемые вопросы

Что такое WhatsApp Coexistence?
WhatsApp Coexistence - это функция, которая позволяет предприятиям использовать как приложение WhatsApp Business, так и WhatsApp Cloud API на одном и том же номере телефона одновременно. Сообщения синхронизируются в реальном времени между обеими платформами, позволяя вам сохранить личный подход мобильного приложения, одновременно используя автоматизацию и масштабируемость API.
Мне нужно менять номер телефона?
Нет! Одним из ключевых преимуществ WhatsApp Coexistence является то, что вы сохраняете свой существующий номер телефона. Вам не нужно получать новый номер или изменять существующий. Процесс интеграции на основе QR-кода связывает ваш текущий номер WhatsApp Business App с Cloud API.
Я потеряю историю чата?
Нет, вы можете сохранить до 180 дней истории чата во время интеграции. Когда вы связываете ваше приложение WhatsApp Business с Cloud API через встроенный процесс регистрации, у вас есть возможность импортировать последние шесть месяцев истории разговоров и контактов. Однако это должно быть сделано в течение 24 часов после завершения процесса интеграции.
Как работает тарификация?
WhatsApp Coexistence позволяет использовать гибридную модель тарификации. Сообщения, отправленные вручную из WhatsApp Business App, остаются 100% бесплатными. Сообщения, отправленные через Cloud API (включая автоматические сообщения, шаблоны и рассылки), следуют модели тарификации Meta на основе разговоров. Это позволяет оптимизировать затраты, используя бесплатные сообщения приложения для личных 1:1 разговоров и сообщения API для масштабирования и автоматизации.
Каковы технические ограничения?
Основные технические ограничения: 1) Пропускная способность ограничена 20 сообщениями в секунду (MPS) для аккаунтов сосуществования, по сравнению с более высокими лимитами для автономных аккаунтов API; 2) Вы должны открывать WhatsApp Business App по крайней мере раз в 13-14 дней, чтобы поддерживать соединение синхронизации; 3) Некоторые функции приложения отключены, включая синхронизацию групповых чатов, исчезающие сообщения, медиафайлы для однократного просмотра и редактирование/отмену сообщений.
Доступно ли это во всех странах?
Нет, WhatsApp Coexistence находится в фазе поэтапного запуска. По состоянию на январь 2026 года недоступно в определенных регионах, включая Нигерию и Южную Африку. Бизнесы с телефонными номерами из этих стран в настоящее время не имеют права и должны выбрать между автономным приложением или автономным API.
Сколько времени занимает миграция?
Процесс миграции спроектирован так, чтобы быть быстрым и бесшовным. Встроенный процесс регистрации на основе QR-кода обычно занимает 5-10 минут. Если вы решите импортировать историю чата, это будет происходить в фоновом режиме. Весь процесс может быть завершен менее чем за 15 минут без каких-либо перерывов в ваших текущих операциях WhatsApp.
Что происходит, если я прекращу его использовать?
Вы можете прекратить использование Cloud API в любое время. Ваше приложение WhatsApp Business будет продолжать работать нормально с вашим номером. Однако вы потеряете доступ к автоматизации, функциям многоагентности и интеграции с CRM, которые обеспечивал API. Вы не сможете повторно активировать сосуществование для этого номера, пока не свяжетесь со службой поддержки Meta и не пройдете новый процесс интеграции.
